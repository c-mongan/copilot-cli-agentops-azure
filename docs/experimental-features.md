# Experimental Features

The core product is Copilot CLI metadata telemetry through a local Collector into Azure/Grafana.

These are experimental:

- benchmarks
- saved views
- custom event emit/import
- ask-context
- policy, permission-friction, lineage, and MCP dashboards
- Codex wrapper support
- MCP analyst workflows
- alert recommendations
- actioners

Use:

```bash
agentops experimental <command> [...]
```

Old top-level commands still print migration guidance where reasonable.
